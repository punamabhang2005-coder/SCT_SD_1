Temperature Converter (Python CLI Application)

---

## 📖 Project Description

This project is developed as part of the SkillCraft Technology Internship.

The Temperature Converter is a command-line Python application that allows users to convert temperature values between:

1. Celsius to Fahrenheit
2. Celsius to Kelvin
3. Fahrenheit to Celsius
4. Fahrenheit to Kelvin
5. Kelvin to Celsius
6. Kelvin to Fahrenheit

The program takes user input, performs the selected conversion, and displays the result instantly.

---

## 🛠 Technologies Used

- Python 3
- VS Code
- Command Line Interface (CLI)

---

## ⚙️ How It Works

1. The program displays a menu of temperature conversion options.
2. The user selects a choice (1–6).
3. The user enters the temperature value.
4. The program calculates and displays the converted result.
